# yande.re_crawler
![](https://img.shields.io/hexpm/l/plug.svg)

**My very first crawler using python.**

##Requirements

- Python 3.5+
- BeautifulSoup 4
- Pillow
- requests

##Usage

Create a folder named 'download'

`python yande_v3.py`

Type your URL form [yande.re](https://yande.re)

The image will be downloaded to folder 'download'
